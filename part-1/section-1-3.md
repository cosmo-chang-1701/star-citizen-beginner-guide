# 1.3：我的第一艘船該怎麼選？當前的「起跑線」指南

你的第一艘船不僅是你在宇宙中的第一個家，更是你累積財富的工具。隨著遊戲機制的深化，入門船的選擇已從簡單的「戰鬥或送貨」演變為「職業路徑的選擇」。

以下是我們為你篩選的三個最佳入門方案：

### 1. 經濟首選：RSI Aurora Mk II

這是繼承了經典極光系列精神的新一代入門機型，也是目前預算最友善的選擇。

| 特點 | [RSI Aurora Mk II (極光 Mk II)](https://robertsspaceindustries.com/en/pledge/Packages/Citizen-Starter-Pack) |
| :--- | :--- |
| **定位** | **多功能通用入門機** |
| **核心優勢** | **高性價比、模組化空間**。 Mk II 優化了內部空間，不僅保留了方便的**生活床位**，更強化了外部掛載點。 |
| **適合玩法** | 基礎送貨、輕型戰鬥、快遞任務。 |
| **新手評語** | 如果你想用最低的成本進入宇宙，並擁有隨時隨地離線休息的權利，Mk II 是你的不二之選。 |

---

### 2. 全能王者：Aegis Avenger Titan (復仇者 泰坦)

儘管時光流逝，泰坦在目前的宇宙環境中依然穩坐「新手村霸主」的寶座。

| 特點 | [Aegis Avenger Titan (復仇者 泰坦)](https://robertsspaceindustries.com/en/pledge/Packages/Duelist-Starter-Pack) |
| :--- | :--- |
| **定位** | **戰鬥/運輸雙修萬金油** |
| **核心優勢** | **全能性**。它擁有比基礎船強得多的火力、靈活的機動性，以及一個 8 SCU 的內部貨艙。尾部的**坡道門**讓你在執行地堡任務或搬運物資時極其方便。 |
| **適合玩法** | 中級賞金獵人、武裝貿易、地堡突擊、多階段戰術任務。 |
| **新手評語** | 如果你願意多花大約 20 美元，泰坦能讓你跳過初期的尷尬期，直接參與宇宙中大多數的高收益活動。 |

---

### 3. 專業起點：Drake Golem (德雷克 魔像)

這是專為「立志成為專業人士」的新手設計的全新選擇。

| 特點 | [Drake Golem (德雷克 魔像)](https://robertsspaceindustries.com/en/pledge/Packages/Miner-Starter-Pack) |
| :--- | :--- |
| **定位** | **工業/採礦專業入門機** |
| **核心優勢** | **專業化工具**。它是第一艘整合了小型採礦雷射與物資粉碎裝置的入門船，讓玩家從第一天起就能參與宇宙最穩定的經濟活動：採礦。 |
| **適合玩法** | 手機採礦、精煉物資、工業調查。 |
| **新手評語** | 不想打打殺殺？Golem 是你進入星際工業界的門票，它雖然醜陋笨重，但卻是致富最快的工具之一。 |

---

### 總結建議：你該選哪一艘？

- **預算有限，想體驗一切：** 選擇 **Aurora Mk II**。
- **想要最強的戰鬥力與靈活性：** 選擇 **Avenger Titan**。
- **明確目標就是「賺錢與工業」：** 選擇 **Drake Golem**。

**小提醒：** 所有的入門船都可以在官網透過 **「Ship Upgrades (CCU)」** 功能補差價升級成更好的船。如果你先買了 Mk II 玩一段時間後發現自己喜歡採礦，你可以直接把它升級成 Golem，而不需要重新購買遊戲包。
