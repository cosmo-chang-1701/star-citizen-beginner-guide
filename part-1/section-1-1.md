# 1.1：我的電腦跑得動嗎？硬體需求溫馨提醒

在所有步驟開始之前，最現實的問題是：你的電腦能應付這個被稱為「硬體殺手」的遊戲嗎？老實說，《星際公民》對硬體的要求非常高。官方網站上的「最低需求」僅僅代表「能打開遊戲」，離「順暢遊玩」還有很長一段距離。

以下是我們根據大量玩家的實際經驗，為你整理的「**真正的新手入門標準**」：

#### 關鍵中的關鍵：SSD (固態硬碟)

- **要求**： **官方強制性要求，HDD 已不被支援。**
- **原因**： 《星際公民》採用資料流技術即時讀取海量模型與貼圖。使用傳統硬碟 (HDD) 會導致嚴重卡頓、貼圖消失（例如掉進地底下）甚至遊戲崩潰。
- **建議**： 必須安裝在 SSD 上，且至少預留 150GB 空間。強烈建議使用 **NVMe M.2 SSD**，其讀取速度對於 2026 年的遊戲版本（如 Alpha 4.x）至關重要。

#### 記憶體 (RAM)：32GB 是流暢的基準

- **要求**： **最低 16GB，強烈建議 32GB 或以上。**
- **原因**： 雖然官方最低門檻為 16GB，但在大型城市（如 Lorville 或 Area18）中，記憶體佔用量經常突破 20GB。16GB 系統會頻繁調用分頁檔案，導致瞬間卡頓。
- **建議**： 32GB 是目前社群公認的標配。如果你有進行多工處理或直播的需求，64GB 也是合理的投資。

#### 中央處理器 (CPU)：L3 快取的勝利

- **指令集要求**： CPU 必須支援 **AVX, AVX2, 與 FMA3** 指令集（Intel Haswell / AMD Excavator 以後架構）。
- **CPU 的重要性**： 《星際公民》極度依賴 CPU 處理複雜的物理實體。目前 **AMD 的 X3D 系列**（如 7800X3D, 9800X3D）憑藉巨大的 L3 快取，在遊戲中展現出壓倒性的效能優勢。
- **GPU 的分擔技巧**：
  > **重要設定**： 在遊戲設定中，將「品質」(Quality) 選項設定為「高」(High) 或以上。這會促使遊戲將更多渲染任務從 CPU 卸載到 GPU，在多數情況下反而能提升幀數。

#### 顯示卡 (GPU) 與 Vulkan API

- **驅動與技術**： 顯示卡必須支援 **Vulkan 1.2** 與 DirectX 12。
- **顯示卡建議**： 建議擁有至少 **12GB VRAM** 的顯示卡以應付高解析度貼圖。對於 1440p 或 4K 玩家，RTX 4070 或更新的 **RTX 50 系列**是理想選擇。

#### 效能提升關鍵：啟用 XMP / EXPO

- **為什麼重要？** 確保你的記憶體運行在標稱頻率（如 6000MHz 或更高）。未開啟此功能會導致記憶體以預設低速運行，這在《星際公民》這種資料吞吐量巨大的遊戲中，會造成顯著的效能損失。

---

### 軟體調校：2026 年的最佳設定

#### 遊戲內設定 (Options -> Graphics)

- **品質 (Quality)**：**High** 或 **Very High**。
- **星球體積雲 (Planet Volumetric Clouds)**：最吃效能！建議設為 **Medium** 或 **Off**。
- **升級技術 (Upscaling)**：這是提升 FPS 的利器。NVIDIA 玩家開啟 `DLSS 3.5+`，AMD/Intel 玩家開啟 `FSR 3+` 或 `XeSS`，並設為 **Quality** 模式。
- **Vulkan API**：在 2026 年的版本中，Vulkan 已成為穩定且建議的預設選項，能有效利用多核心 CPU 效能。

#### 驅動程式與系統優化

- **著色器快取 (Shader Cache Size)**：NVIDIA 玩家請在控制面板將其設為 **10 GB** 或 **無限制**，這能大幅減少初次進入新場景時的卡頓。
- **虛擬記憶體 (Pagefile)**：如果你只有 16GB/32GB RAM，請確保在 SSD 上設定了至少 16GB 的手動分頁檔案，以防止「記憶體不足 (Out of Memory)」崩潰。
- **停用全螢幕最佳化**：在 `StarCitizen.exe` 內容中勾選此項，能減少某些系統上的顯示延遲。

### 組裝建議：2026 年三套推薦配置

#### 「極光 MR」— 1080p 經濟實惠款

- **CPU**：AMD Ryzen 7 5700X3D
- **GPU**：NVIDIA GeForce RTX 4060 (8GB) 或 RTX 3060 (12GB)
- **RAM**：32GB DDR4
- **SSD**：1TB NVMe Gen3/4

#### 「彎刀黑」— 1440p 效能主流款

- **CPU**：AMD Ryzen 7 7800X3D (目前性價比之王)
- **GPU**：NVIDIA RTX 4070 SUPER / AMD RX 7900 GRE
- **RAM**：32GB DDR5-6000
- **SSD**：2TB NVMe Gen4

#### 「克拉克」— 4K / 旗艦無妥協款

- **CPU**：AMD Ryzen 7 9800X3D (或旗艦級 9950X3D)
- **GPU**：NVIDIA GeForce **RTX 5080 / 5090**
- **RAM**：64GB DDR5-6400+
- **SSD**：4TB+ NVMe Gen5
