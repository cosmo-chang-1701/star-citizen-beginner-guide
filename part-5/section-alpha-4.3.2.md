# Alpha 4.3.2：破碎邊境 (Fractured Frontiers)

{% embed url="https://www.youtube.com/watch?v=R9Z8BImk_tM" %}

公民，歡迎來到 Alpha 4.3.2。官方命名為「破碎邊境」（Fractured Frontiers），這不僅是一次更新，更是過去數個版本醞釀已久的劇情高潮。開發者表示：「一切皆源於此」。

本指南將協助你應對由頭號通緝犯 Amelia Boyd 引發的星系動亂，並指導你如何在「邊境戰士最終章」中與全伺服器玩家協力追獵北極星級護衛艦。

## 1. 劇情背景：恐怖的餘波

這次風暴的中心是 Amelia Boyd，她偷走了一艘強大的北極星級 (Polaris-class) 護衛艦，將派羅 (Pyro) 星系攪得天翻地覆。

*   **敘事連結**：從「拯救斯坦頓」的切片者攻擊，到「為派羅而戰」與「生死供應」，所有行動最終彙整於此。
*   **長期目標**：找到並逮捕（或摧毀）Amelia Boyd，這是所有行動的最高指導原則。
*   **動態敘事**：你的選擇與行動將產生漣漪效應，直接影響未來遊戲世界的星際政治格局。

## 2. 戰區分析：陣營與勢力抉擇

在混戰中，選對邊站是成功的第一步。玩家需要從兩個對立陣營中選擇加入，為其賺取貢獻點。

*   **敵對勢力：邊境戰士 (Frontier Fighters)**：由 Amelia Boyd 領軍，手握被竊的北極星護衛艦，威脅著整個星系的穩定。
*   **繁榮公民 (CFP)**：主張在派羅建立「法律與秩序」，背後疑似有大企業支持，立場偏向守法中立。
*   **獵頭幫 (HH)**：犯罪色彩濃厚的部落式幫派，適合享受法外狂徒自由的玩家。

## 3. 事件檔案：邊境戰士最終章

「邊境戰士最終章」是一個持續數週的動態事件，分為三個核心階段：

| 階段 | 任務名稱 | 任務目標簡述 | 階段性獎勵 |
| :--- | :--- | :--- | :--- |
| **階段一：偵察** | 定點清除 / 失蹤人員 | 執行斬首打擊或回收關鍵數據板，削弱敵方指揮系統。 | 所選陣營風格的完整服裝 (含裝甲)。 |
| **階段二：圍攻** | 艦船伏擊 / 摧毀再生點 | 伏擊敵方巡邏隊或滲透地面設施摧毀醫療物資。 | 陣營主題的灰貓 MTC 地面載具塗裝。 |
| **階段三：對決** | 追獵北極星 | 參與全伺服器聯合行動，摧毀被竊的北極星級護衛艦。 | 鐵砧箭矢與十字軍聖靈的陣營主題塗裝。 |

> **專家提示**：任務階段皆可在斯坦頓或派羅進行。新玩家建議在斯坦頓進行活動，以降低與其他資深玩家發生衝突的風險。建議組隊分享任務，雖然獎金平分，但貢獻點是每位玩家完整獲取。

## 4. 戰略深潛：代號「追獵北極星」行動

這是全伺服器共享的大型 PvE 任務。目標是合作追擊並摧毀不斷移動的北極星護衛艦。

*   **伺服器級協調**：強烈建議利用全域頻道建立指揮系統，將艦隊分為：
    *   **主力輸出**：負責對北極星船體進行重火力打擊。
    *   **護航清剿**：清理敵方的護航戰鬥機。
    *   **攔截攔門**：使用量子攔截船防止目標逃跑。
    *   **後勤支援**：提供戰地修理、加油與醫療。

## 5. 軍械庫推薦：邊境作戰資產

選對工具是勝利的保障。Alpha 4.3.2 引入了數款專為此次大戰設計的新船：

| 任務角色 | 推薦艦船 | 最佳配裝策略 |
| :--- | :--- | :--- |
| **主力輸出** | **Anvil Paladin** (新) | 重型炮艇，適合對大型目標進行持續彈幕攻擊。 |
| **護航清剿** | **Esperia Stinger** (新) | 復刻版剜度重戰，高耐久與致命火力，適合清理敵機。 |
| **混合戰鬥** | **Grey's Market Shiv** (新) | 拼湊風格的重戰，性價比高，適合伏擊。 |
| **戰術支援** | **RSI Mantis** | 必備的量子攔截能力，防止目標撤離。 |

> **專家提示：FPS 武器平衡**
> 本版本大幅增強了輕機槍 (LMG) 的壓制力。實彈狙擊步槍現在可穿透能量護盾，適合在地面任務中執行精確拔點。

## 6. 周邊更新：結構回收與經濟平衡

*   **結構回收重做**：引入「最佳位置 (Sweet Spots)」機制，正確對齊可獲得高達 7 倍的回收速度加成。聖盾回收者 (Reclaimer) 與 Drake 禿鷲 (Vulture) 現在有了更清晰的職業定位。
*   **經濟調整**：派羅星系現在是交易廢料 (Scrap) 利潤最高的地方，旨在鼓勵跨星系貿易。

---

**結語**：Alpha 4.3.2 的「破碎邊境」不僅是技術層面的壓力測試，更是《星際公民》轉向「連續劇式」敘事的重要轉折。我們在這次大戰中的每一次協作，都在定義未來宇宙的樣態。
