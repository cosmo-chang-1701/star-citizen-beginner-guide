# Alpha 4.3.1：海伯利安計畫 (Project Hyperion)

{% embed url="https://www.youtube.com/watch?v=pq7XEWatINA" %}

公民，歡迎來到 Alpha 4.3.1。這是 Onyx 設施故事線的高潮篇章。隨著「海伯利安計畫」的正式啟動，我們終於能深入最神祕的 B 號站點 (Site B) 科學翼，揭開喬里特博士那扭曲遺產的真相。

本指南將從背景敘事、前置任務、核心解謎到最終撤離，為你提供全方位的深度攻略。

## 1. 敘事核心：B 號站點與海伯利安計畫

Onyx 設施不單是個任務地點，它是一個充滿科幻恐怖氣氛的敘事舞台：

*   **ASD 的陰影**：先進系統部 (ASD) 表面研究再生危機，實則進行非法的人類進化實驗。
*   **喬里特博士 (Dr. Jorrit)**：陰謀的核心策劃者，B 號站點是他最扭曲實驗的所在地。
*   **海伯利安計畫**：涉及基因改造與高級再生技術的禁忌研究。
*   **重組體 (Recombinant)**：任務最終生成的特殊物品，是解開未來故事章節的關鍵鑰匙。

## 2. 任務檔案：海伯利安計畫合約

這不是普通的合約，而是對玩家技巧與耐心的終極考驗。

*   **解鎖條件**：需先完成霍克羅事務所 (Hockrow Agency) 的 7 個前置調查合約（涉及回收工程、人事、研究等數據）。
*   **合約報酬**：高達 **677,000 - 700,000 aUEC**，是目前最高收益的 FPS 任務之一。

> **專家提示：裝備建議**
> *   **武器**：推薦組合式配置。主武器建議選用高 DPS 的突擊步槍或 Alpha 4.3.1 新增的 **VOLT "Fresnel" 輕機槍** 以壓制重甲敵人。
> *   **工具**：必備帶有牽引光束的**多功能工具**，以及足夠的**醫療凝膠 (MedGel)** 資源。

## 3. 滲透指南：工程翼的三大謎題

要進入 B 號站點，必須先破解工程翼的環境系統：

1.  **通風管道謎題**：操控主面板與多層結構中的閥門，依正確順序引導氣體流動。
2.  **冷凝器謎題**：在限時內校準分散在不同樓層的終端陣列，為反應堆降溫。
3.  **燃料反應堆謎題**：最具挑戰性的環節。需安裝保險絲與調節器，並在啟動程序的數分鐘內防禦多波敵人的猛攻。

**關鍵組件獲取**：
*   **試劑代碼 (Reagent Code)**：位於工程翼動力核心區的終端。
*   **催化劑 (Catalyst)**：從指定地點獲取 XTL、PWL 或 RGL 三種催化劑之一。

## 4. 深入 B 號站點 (科學翼)

這是設施中最危險的區域，設計上充滿了壓迫感：

*   **尋找隱藏入口**：穿過研究翼的貨物檢查區與中轉站即可抵達。
*   **環境危害**：注意「不歸路」的垂直設計（一旦跳下天井便無法回頭）以及局部的重力異常。
*   **多方交戰**：除了 ASD 衛兵與重甲兵，還會遇到生物敵人 **科皮恩 (Kopions)**。利用 **QR 部隊** 與 ASD 之間的敵對關係進行「漁翁得利」是高階戰術。

## 5. 終極挑戰：執行海伯利安實驗

抵達位於最深處的喬里特實驗室後，真正的考驗開始：

1.  **實驗室准入**：利用搜刮到的密碼解鎖實驗室大門。
2.  **運行實驗**：在中央控制台輸入試劑代碼並插入催化劑。
3.  **獲取結果**：完成模擬序列後，下載數據並獲得**重組體 (Recombinant)**。
4.  **撤離戰鬥**：返回中央天井呼叫主電梯。準備迎接最後一場激烈的電梯平台防守戰。
