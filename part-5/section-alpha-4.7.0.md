# Alpha 4.7.0：製造、物流與尼克斯之戰 (Crafting & Logistics)

{% embed url="https://www.youtube.com/watch?v=sX4_Q6AuqSI" %}

公民，歡迎來到 Alpha 4.7.0。這是關於「生產力」與「星系擴展」的里程碑式更新。它不僅引入了《星際公民》經濟體系的關鍵環節——初級製造系統，更對尼克斯 (Nyx) 星系進行了大規模的深度開發。

本指南將協助你掌握從原材料到成品的製造流程，並教你如何在尼克斯星系的資源爭奪中佔得先機。

## 1. 初級製造與藍圖 (Crafting Tier 0)

這是 4.7.0 的核心更新，標誌著玩家從單純的消費者轉變為生產者：

*   **製造機 (Fabricator)**：首個可操作的工業設施，現已部署在特定的空間站、前哨站或大型旗艦上。
*   **藍圖系統 (Blueprints)**：玩家可透過任務獎勵、廢墟掃描或從敵對勢力手中獲取各種物品、武器甚至飛船組件的製造藍圖。
*   **材料循環**：利用從採礦獲得的原礦或從打撈獲得的回收金屬，配合對應藍圖即可製造出高品質的裝備。

## 2. 物流與技術革命 (Logistics & Tech)

*   **背包重構與區域掠奪 (Inventory 2.0)**：
    *   **紙娃娃介面**：全新的 UI 讓裝備管理更直觀。
    *   **一鍵堆疊與區域掠奪**：大幅提升了在危險區域（如地堡）收集物資的效率。
*   **交通系統技術預覽 (Transport Overhaul)**：首先在 Area 18 實施的新一代交通系統底層技術，旨在解決電梯與輕軌的穩定性問題。

## 3. 尼克斯星系擴展 (Nyx Expansion)

隨著 Nyx 星系的進一步開發，多個全新的地點與勢力正式加入：

*   **人民服務站 (People's Service Stations)**：由人民聯盟管理的新型社交樞紐，提供補給服務並可設置為重生點。
*   **新勢力：利爪蠑螈 (Claw Salamanders)**：活躍於尼克斯的小行星帶勢力，擁有獨特的「風塵者風格」裝束，是玩家在探索中需要應對的新威脅。
*   **破碎者站點行動 (Operation Breaker Station)**：玩家可以參與圍繞著基格帶附近廢棄拆解站的爭奪戰。

## 4. 戰鬥平衡與新武器

*   **護甲門檻 (Armor Threshold)**：物理化護甲系統的初步實現。小口徑武器難以對重裝甲目標造成有效結構損傷。
*   **組件化雷達與瞄準輔助**：雷達品質現在直接影響探測距離與瞄準輔助的精準度。
*   **新武器**：**MaxOx NN15 中子炮** (Size 3) 正式實裝，為中型船艦提供了強大的火力選擇。

## 5. 新增載具與自定義

*   **RSI Aurora Mk II**：經典入門機型的現代化翻新版本，提升了視覺細節與組件訪問性。
*   **Greycat UTV**：新型小型多功能載具，專為地面物流設計。
*   **角色自定義**：新增了 6 款全新髮型，並優化了 FPS 武器的藝術細節。

| 更新重點 | 核心影響 | 建議行動 |
| :--- | :--- | :--- |
| **製造系統** | 開啟玩家驅動的生產經濟。 | 收集採礦與打撈材料，尋找初級藍圖。 |
| **尼克斯擴展** | 增加高風險、高回報的探索區域。 | 前往人民服務站，與當地勢力建立聲望。 |
| **護甲門檻** | 強化了船艦的戰術分工。 | 針對重型目標改用重型武器或魚雷。 |

> **專家提示**：製造系統需要穩定的原材料供應。如果你是戰鬥玩家，建議與採礦或打撈玩家合作，或保留你在戰鬥中繳獲的敵艦殘骸，這些都是珍貴的製造資源。
