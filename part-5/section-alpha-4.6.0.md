# Alpha 4.6.0：列夫斯基的生命線 (Lifeline for Levski)

{% embed url="https://www.youtube.com/watch?v=OPp6dd2WSpI" %}

公民，歡迎來到 Alpha 4.6.0。這次更新被稱為「列夫斯基的生命線」，它不僅帶來了玩家期盼已久的夜視系統，更透過一場席捲 Nyx 星系的敘事活動，將宇宙的生存壓力推向了新的高度。

本指南將協助你掌握 4.6.0 的核心功能更新，並教你如何在「莫利納黴菌」感染危機中贏得專屬獎勵。

## 1. 光放大系統 (Night Vision / LAMP)

這是《星際公民》歷史上最重要的 QoL（生活品質）更新之一：

*   **地形夜視 (LAMP)**：駕駛艙現在配備了光放大系統。在行星黑暗面飞行時，開啟夜視功能（通常與掃描模式連動）可清晰識別地形起伏，徹底終結了「盲飛撞山」的時代。
*   **個人夜視**：特定的頭盔現在也支持步兵級夜視，提升了夜間地面行動的戰術價值。

## 2. RSI Aurora 金標化 (Gold Standard)

作為無數公民的起點，Aurora 系列迎來了全面的物理化升級：

*   **物理化組件**：發電機、冷卻器與護盾產生器現在變成了可見、可觸達的實體組件，支持手動維修。
*   **視覺優化**：內飾細節與 VTOL 推進器的邏輯重構，使其更符合當前的物理引擎標準。

## 3. 工業訂單與經濟調整

隨著開發節奏轉向「雙月更」模式，本版本強化了工業玩家的穩定收益：

*   **大宗物資合約**：由 Shubin 或 Greycat 等公司發布。玩家需交付特定數量的精煉材料，提供穩定的收入曲線與企業聲望。
*   **物流效率提升**：優化了精煉廠的數據交互，減少了在大宗交易時的延遲感。

## 4. 敘事活動：清除莫利納黴菌 (Clearing the Air)

列夫斯基 (Levski) 站因空氣過濾器缺陷爆發了致命黴菌感染，引發了全伺服器的救援行動：

*   **救援合約**：玩家可作為 CureLife 承包商運送醫療物資，或進入受感染區域執行環境淨化任務（需穿戴重型生化防護服）。
*   **專屬獎勵**：完成活動可獲得 CureLife 主題護甲及限定船艦塗裝。

| 更新重點 | 核心影響 | 建議行動 |
| :--- | :--- | :--- |
| **夜視系統** | 提升全天候飛行安全性。 | 將夜視功能綁定至易操作按鍵。 |
| **Aurora 金標** | 提升初級船隻的可玩性。 | 練習手動更換損壞的船艦保險絲。 |
| **黴菌清除活動** | 獲取限時醫療主題裝備。 | 準備好重型防護服，前往 Nyx 支援救援。 |

> **專家提示**：夜視系統 (LAMP) 在低能見度降落時非常有效，但注意強光源（如空間站燈光）可能會造成視覺過載，建議在接近明亮區域時暫時切回普通模式。

---

**結語**：Alpha 4.6.0 證明了即使在沒有大規模地圖擴張的情況下，透過核心機制的完善與深刻的敘事活動，也能讓宇宙煥發新生。
