# Alpha 4.3.0：Onyx 設施 (Onyx Facilities)

{% embed url="https://www.youtube.com/watch?v=MCJlD2Meuek" %}

公民，歡迎來到 Alpha 4.3.0。這次更新為我們揭開了「Onyx 計畫」神祕的面紗，引入了散落在斯坦頓星系各處的廢棄地下實驗室——Onyx 設施。

本指南將協助你深入這些充滿環境危害與敵對勢力的地牢，並教你如何安全地帶走那些令人垂涎的頂級戰利品。

## 1. Onyx 計畫背景與設施結構

這些設施曾是聯合科學發展部 (ASD) 為解決「再生危機」而設立的實驗室。由於不人道的實驗引發了嚴重的事故，如今這裡只剩下輻射外洩、電力不穩與致命的威脅。

*   **入口與安全區**：進入設施的唯一途徑是載具機庫。機庫連接著大廳（禁火區），內有醫療床可供休整。
*   **核心區域**：搭乘大廳電梯可前往研究翼 (Research Wing) 與工程翼 (Engineering Wing)。
*   **不確定的遭遇**：設施並非獨立副本。雖然系統會嘗試分散玩家，但你隨時可能與其他探索者或執行不同任務的小隊（如刺殺任務小隊）不期而遇。

## 2. 任務導引：從教學到農耕

建議新手先接取 Hockrow 事務所的合約，按部就班熟悉環境：

1.  **引導任務 (Arken Mallor)**：強制性的教學關卡，要求你跑遍研究翼與工程翼的核心區域。這是熟悉掩體位置與戰利品刷點的最佳機會。
2.  **重複性任務**：解鎖後可穩定賺取 ASD 安全硬碟，用於在 Wikelo 商人處兌換保底獎勵。
3.  **多樣化挑戰**：設施內可能同時存在多個不同合約的目標，這讓玩家間的互動（合作或衝突）變得極其複雜。

## 3. 戰利品搜刮指南：Geist 裝甲與新武器

我們來到 Onyx 的首要目標通常是那些稀有的裝備：

*   **Geist 裝甲套裝**：部件分散在不同的區域，需要耐心收集。
*   **ASD 稀有變體**：全黑色的 ASD 版本 Geist 裝甲掉落率僅約 1.2%，建議保持平常心，視為意外驚喜。
*   **VOLT 'Zenith' 狙擊槍**：全新的雷射狙擊槍可在設施內（如工程翼懸臂走道）免費獲取。

## 4. 生存戰術：應對 PvE 與 PvP 威脅

活著帶走戰利品才是真正的成功：

*   **控制而非殲滅 (PvE)**：NPC 敵人是無限重生的。對於獨行玩家，強烈建議使用**抑制器（消音器）**進行潛行操作，只清除阻礙路徑的敵人。
*   **心理博弈 (PvP)**：電梯口和狹窄走廊是高風險區域。
    *   **停、聽、看**：出電梯前先聆聽外面的動靜。
    *   **識別玩家**：目前設施內的 NPC 不會配戴背包，這是一個快速區分玩家與 NPC 的關鍵特徵。
    *   **聲音辨識**：學習區分 NPC 制式武器與玩家特殊武器（如雷射武器）的聲音。

## 5. 推薦配置與撤退邏輯

*   **個人配置**：帶抑制器的主武器、足夠的醫療筆與輻射筆。推薦攜帶多功能工具（配備挖礦與維修模組）。
*   **載具選擇**：推薦「快進快出」的船隻，如 Avenger Titan 或 Cutter。擁有床鋪的載具方便在需要時更換伺服器。

> **專家提示**：學會風險管理。你在設施待得越久，風險越高。NPC 會消耗你的彈藥，其他玩家進入的機率也會增加。適時的「閃擊戰」與成功撤退，遠勝過貪心導致的全盤皆輸。

---

**結語**：Alpha 4.3.0 的 Onyx 設施挑戰玩家的心態與耐心。這是一個將敘事與高難度地牢玩法完美結合的區域。無論是潛行尋寶還是團隊強攻，每一次成功的逃離都是對你戰術素養的肯定。

公民，保持警惕，願你在 Onyx 的黑暗中收穫滿滿！
