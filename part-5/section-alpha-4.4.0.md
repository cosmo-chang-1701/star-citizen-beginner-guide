# Alpha 4.4.0：歡迎來到 Nyx (Welcome to Nyx)

{% embed url="https://www.youtube.com/watch?v=ilDmGyHLQNg" %}

公民，歡迎來到 Nyx。隨著 Alpha 4.4.0 版本的實裝，我們見證了持續宇宙（PU）的一次關鍵轉折。這不僅僅是地圖的擴張，而是開發團隊首次將「背景敘事」與「核心玩法」進行了深度綁定。

過去我們熟悉的斯坦頓（Stanton）是企業的遊樂場，派羅（Pyro）是無政府的荒野，而現在回歸的 Nyx 星系，則是用來連接這兩者，並正式引入剜度（Vanduul）威脅的戰略樞紐。這份指南將協助各位理解新版本的運作邏輯，從地理環境、派系鬥爭到貿易機會，我們逐一拆解。

## Nyx 星系：環境與地緣政治重構

Nyx 星系的重頭戲在於德拉瑪（Delamar）小行星與萊夫斯基（Levski）的回歸。對於老玩家來說，這是歷史記憶的修正；對於新公民來說，這裡是感受「生存主義」美學的最佳場所。

| 地點 / 區域                     | 關鍵特徵與敘事意義                                                           | 環境細節與氛圍                                                                                  |
| :------------------------------ | :--------------------------------------------------------------------------- | :---------------------------------------------------------------------------------------------- |
| **萊夫斯基 (Levski)**           | 反威權主義堡壘、「人民聯盟」據點。這裡沒有企業的整潔，只有資源匱乏下的堅韌。 | 視覺上呈現拼湊風格；聽覺上充滿了鑽探井的風聲與市集的嘈雜對話，強調孤立感。                      |
| **任務給予者：Recco Battaglia** | 該區域採礦與資源開採的樞紐。她的回歸標誌著獨立礦工經濟圈的重啟。             | 對話風格務實直接。任務多涉及資源保護與領地爭奪，反映了當地的經濟現實。                          |
| **冰川環 (Glaciem Ring)**       | 環繞恆星的密集小行星帶，隱藏著大量廢棄的 QV 物流服務站。                     | 「誓不兩立」行動的主要舞台。這裡是逃避官方視線的避風港，也是各類殘骸的聚集地。                  |
| **破碎之刃 (Shattered Blade)**  | 位於冰川環內的戰場遺跡，保留了剜度襲擊的直接證據。                           | 玩家在此進行「戰場考古」，透過掃描與調查，將外星威脅從傳說具象化為現實證據。                    |
| **跳躍點 (Jump Points)**        | 連接斯坦頓、派羅與 Nyx 的通道，不再只是快速旅行機制。                        | 充當了威脅（剜度）與腐敗（ASD）擴散的物理管道，玩家穿越時實際上是在切換完全不同的社會契約區域。 |

## 核心衝突：「誓不兩立」行動與派系角力

這次更新的核心敘事圍繞著「誓不兩立」（Sworn Enemies）行動展開。我們必須理解三方勢力的動機，才能在任務中做出正確判斷。這條故事線揭露了為了從「重生危機」（Regen Crisis）中尋找出路，人類企業是如何越過道德底線的。

| 勢力名稱                     | 角色定位                           | 核心動機與行為模式                                                                                                                         |
| :--------------------------- | :--------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------- |
| **InterSec**                 | 私人安保與調查承包商（玩家的雇主） | **目標**：調查非法異種科技流動。<br>**行為**：因 UEE 無力監管邊境，InterSec 依賴高信譽度的玩家（獨立承包商）來揭露 ASD 的陰謀。            |
| **聯合科學與發展公司 (ASD)** | 激進的科學實體（主要的道德反派）   | **目標**：解決重生印記退化的危機。<br>**行為**：從派羅的生物實驗轉向購買被禁的剜度科技。為了拯救人類的「永生」技術，不惜與人類的天敵交易。 |
| **剜度 (Vanduul)**           | 活躍的生存威脅與技術源頭           | **目標**：滲透與破壞。<br>**行為**：不再是模擬裡的貼圖，而是具有侵略性戰術 AI 的實體。他們是 ASD 技術的賣家，也是攻擊物流鏈的兇手。        |

## 任務流程結構與機制分析

「誓不兩立」是一條多階段的 PvE 調查鏈，它要求玩家具備從空戰到調查的綜合能力。

| 階段                 | 任務類型     | 執行細節與戰術建議                                                                                                                   |
| :------------------- | :----------- | :----------------------------------------------------------------------------------------------------------------------------------- |
| **階段一：信任建立** | **聲望積累** | 玩家需完成 InterSec 的基礎安保合約。只有達到特定軍階（如：Junior rank），才會解鎖關於 ASD 陰謀的核心情報。這體現了情報分級的真實感。 |
| **階段二：調查**     | **現場勘查** | 前往冰川環的 QV 物流站殘骸。利用掃描器識別剜度武器的能量特徵，回收數據板。此階段重點在於「敘事解謎」，而非單純戰鬥。                 |
| **階段三：接觸戰**   | **PvE 空戰** | 這是持續宇宙首次遭遇剜度 NPC 飛行員（駕駛 Scythe 或 Glaive）。他們的 AI 行為與人類海盜截然不同，戰術更具侵略性，旨在清除目擊者。     |

## 綜合武裝與後勤敘事

4.4.0 版本強調「機制即敘事」。所有的功能更新都在為塑造高風險的邊境氛圍服務。

| 新增機制 / 任務                | 運作方式                                          | 戰術與敘事影響                                                                                                 |
| :----------------------------- | :------------------------------------------------ | :------------------------------------------------------------------------------------------------------------- |
| **剜度運輸 (Vanduul Hauling)** | 穿越 Nyx 星系的高價值貨運，標記為「封鎖線突破」。 | 玩家不再是快遞員，而是戰略物資護送者。這是在頂級掠食者（剜度）監視下的航線，隨時可能遭遇伏擊。                 |
| **外部貨運電梯**               | 在空間站外部手動裝卸貨物的物理設施。              | **極致的緊張感**。將原本枯燥的裝貨時間轉化為戰術脆弱點。玩家必須暴露在停機坪上操作，同時雷達可能隨時亮起紅點。 |
| **綜合武裝場景**               | 無縫連接太空壓制與地面/站內 FPS 戰鬥。            | 要求玩家精通多種戰鬥維度。例如：在太空中癱瘓敵艦，隨即強行登陸並利用貨運電梯作為戰術入侵點進行 FPS 掃蕩。      |

## 結論：暴風雨前的寧靜

公民們，Alpha 4.4.0 是一次極具野心的更新。它告訴我們，剜度戰爭不再是遙遠的背景故事，而是正在發生的現實。ASD 的墮落提醒我們，人類內部的貪婪可能比外星人更危險。

在這個版本中，請特別留意你的聲望選擇與物資準備。Nyx 星系是即將到來的全面衝突的風眼，而我們——這些獨立承包商，將決定這場戰爭的走向。

祝各位在 Nyx 星系航行平安。
