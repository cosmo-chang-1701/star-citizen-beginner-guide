# Alpha 4.4.0：歡迎來到 Nyx (Welcome to Nyx)

{% embed url="https://www.youtube.com/watch?v=ilDmGyHLQNg" %}

公民，歡迎來到 Alpha 4.4.0。這是持續宇宙 (PU) 的一次關鍵轉折點。隨著 Nyx 星系的正式實裝，開發團隊首次將深度的「背景敘事」與「核心玩法」進行了全面綁定。

本指南將協助你理解 Nyx 的運作邏輯，從地緣政治、派系鬥爭到全新的「誓不兩立」行動，帶你拆解這個連接斯坦頓與派羅的戰略樞紐。

## 1. Nyx 星系：環境與地緣政治重構

Nyx 星系的重頭戲在於德拉瑪 (Delamar) 小行星與列夫斯基 (Levski) 的回歸。這裡是感受「生存主義」美學的最佳場所。

| 地點 / 區域 | 關鍵特徵與敘事意義 | 環境細節與氛圍 |
| :--- | :--- | :--- |
| **列夫斯基 (Levski)** | 反威權主義堡壘、「人民聯盟」據點。 | 視覺上呈現拼湊風格，強調物資匱乏下的堅韌感。 |
| **Recco Battaglia** | 採礦與資源開採樞紐。 | 任務多涉及資源保護與領地爭奪。 |
| **冰川環 (Glaciem Ring)** | 密集小行星帶，隱藏大量廢棄 QV 站點。 | 「誓不兩立」行動的主要舞台。 |
| **破碎之刃** | 戰場遺跡，保留剜度襲擊的直接證據。 | 玩家可在此進行「戰場考古」與掃描調查。 |

## 2. 核心衝突：「誓不兩立」行動

這次更新的核心敘事揭露了人類企業為了應對「重生危機」而跨越道德底線的真相。

*   **InterSec (玩家雇主)**：私人安保公司，依賴獨立承包商（玩家）揭露非法異種科技流動。
*   **聯合科學與發展公司 (ASD)**：主要的道德反派，為了拯救人類的「永生」技術，不惜與天敵交易被禁的剜度科技。
*   **剜度 (Vanduul)**：不再是背景貼圖，而是具有侵略性 AI 的實體威脅，作為技術賣家與物流鏈破壞者出現。

## 3. 任務流程：從調查到接觸戰

「誓不兩立」是一條多階段的 PvE 調查鏈，考驗玩家的綜合能力：

1.  **聲望建立**：完成基礎安保合約，達到特定軍階以解鎖核心情報。
2.  **現場勘查**：前往冰川環殘骸，利用掃描器識別能量特徵並回收數據板。
3.  **正式接觸**：首次在持續宇宙中遭遇真正的剜度 NPC 飛行員，應對極具侵略性的戰術行為。

## 4. 綜合武裝與後勤機制更新

4.4.0 版本強調「機制即敘事」，所有的功能更新都在塑造高風險的邊境氛圍：

| 新增機制 | 運作方式 | 戰術影響 |
| :--- | :--- | :--- |
| **剜度運輸** | 穿越 Nyx 星系的高價值「封鎖線突破」貨運。 | 玩家成為戰略物資護送者，隨時面臨頂級掠食者的伏擊。 |
| **外部貨運電梯** | 在空間站外部手動裝卸貨物的物理設施。 | 增加了戰術脆弱點，玩家需在雷達威脅下暴露在停機坪操作。 |
| **綜合武裝場景** | 太空壓制與站內 FPS 戰鬥的無縫連接。 | 要求精通多種戰鬥維度，如癱瘓敵艦後直接強行登陸掃蕩。 |

> **專家提示**：在 Nyx 星系，雷達上的紅點可能代表了比海盜危險百倍的對手。請務必在進出空間站時保持高度警戒，並隨時檢查你的外部裝卸進度。
