# Alpha 4.5.0：工程、生存與資源網格 (The Engineering Update)

{% embed url="https://www.youtube.com/watch?v=JgpLAqW1wOU" %}

公民，歡迎來到 Alpha 4.5.0。這次更新標誌著船艦操作進入了一個全新的維度。原本簡單的電力控制已演變為複雜且動態的物理化管理——「工程與資源網格」系統。

本指南將協助你掌握船艦物理化管理的細節。如果你習慣單人駕駛中大型船隻，本版本將迫使你重新思考團隊分工與操作邏輯。

## 1. 工程與資源網格 (Resource Network)

船艦內部的電力、冷卻液與數據現在透過物理化的線路進行傳輸：

*   **工程站 (Engineering Stations)**：中大型船隻配備了專用終端。工程師可精確分配電力給特定推進器或護盾，並可透過「超頻」換取短期爆發性能，但會加劇組件磨損。
*   **物理化損壞**：戰鬥中的損傷可能直接切斷電力管線，導致特定組件（如左側推進器）失效，需要手動更換保險絲或修復。
*   **生命維持系統 (Life Support)**：氧氣與座艙壓力現在與物理組件掛鉤。若模組受損，艙內氧氣將迅速耗盡，需依賴飛行服氧氣。

## 2. 動態火災管理

電力過載或戰鬥損傷會引發真實的火災場景：

*   **火災傳播**：火勢會沿著物理路徑蔓延並消耗氧氣。
*   **應對機制**：玩家需隨身攜帶滅火器。在極端情況下，可透過手動排空特定區域的大氣來窒息火苗。

## 3. 軟死亡 2.0 (Soft Death 2.0)

船隻結構歸零時不再立即爆炸，而是進入「失能」狀態：

*   **失能狀態**：動力室受損但船員有充足時間進行緊急維修或撤離。
*   **登船戰鬥**：此機制大幅提升了 FPS 登船戰鬥的機率，船員必須在失能的艙室內進行最後防禦。

## 4. 裝備與視覺技術更新

*   **物理化頭盔 HUD**：頭盔型號將直接影響視野。重型頭盔防護強但視野窄，探索型頭盔則提供開闊視野與豐富掃描數據。
*   **實驗性 VR 支持**：初步引入 VR 支持，提升了駕駛艙觀察的沉浸感。

| 工程角色職責 | 核心動作 | 關鍵工具 |
| :--- | :--- | :--- |
| **電力管理** | 分配能源負載、超頻關鍵組件。 | 工程終端、保險絲。 |
| **現場修復** | 焊接損壞線路、更換過熱模組。 | 多功能工具 (修理模組)。 |
| **災害控制** | 滅火、管理艙內大氣壓力。 | 滅火器、艙門控制面板。 |

> **專家提示**：在 4.5.0 中，工程師是船艦的「無名英雄」。建議在進行長途量子航行或進入戰鬥前，檢查所有關鍵位置的保險絲儲備，並確保每個船員都知道滅火器的存放位置。
