# Alpha 4.8.0：戰術打擊 (Tactical Strike)

{% embed url="https://www.youtube.com/watch?v=RWQSxN-ZukE" %}

公民，歡迎來到 Alpha 4.8。這次更新被命名為「戰術打擊」（Tactical Strike），不僅帶來了全新的大規模合作任務架構，更對宇宙的後勤體系與保險機制進行了翻天覆地的改革。

如果說 4.4.0 是將威脅具象化，那麼 4.8 則是賦予了公民們應對威脅的深度工具——無論是更精密的燃料補給，還是旗艦級的後勤支持。本指南將協助你快速掌握新版本的核心變革。

## 1. 核心玩法：戰術打擊群 (Tactical Strike Groups)

這是 Alpha 4.8 推出的全新 PvE 任務框架，專為具備一定規模與組織性的玩家小隊設計。它不再是單一的「飛過去打掉目標」，而是一個多階段的戰役流程。

| 任務階段 | 玩法重點 | 戰術建議 |
| :--- | :--- | :--- |
| **階段一：偵察與滲透** | 利用潛行或長程掃描定位目標基地。 | 建議使用具有低信號特徵的飛船，避免過早驚動敵方防空網路。 |
| **階段二：防空壓制 (SEAD)** | 摧毀基地外圍的防空炮塔或干擾塔。 | 這是重型戰機或轟炸機大展身手的時刻，需要護衛機群掩護。 |
| **階段三：物資搶奪/人員營救** | 降落並進行 FPS 戰鬥，完成核心目標。 | 支援船隻需在空中待命，隨時準備進行近接空中支援 (CAS) 或緊急撤離。 |

## 2. 後勤革命：加油 2.0 (Refueling 2.0)

加油機制在 4.8 迎來了「操作員模式」的全面更新，使其從簡單的點擊按鈕變成了具有深度的專業技能。

*   **加油操作員模式 (Refueling Operator Mode)**：專門的加油船（如 Starfarer, Starlite）現在擁有獨立的 UI。操作員需要精確控制加油懸臂，並監控燃料流速與壓力，過高的壓力可能導致接口損壞甚至爆炸。
*   **加油求救信號 (Refueling Beacons)**：如果你在深空斷油，現在可以發布特定的加油請求。目前由 NPC 響應為主，這為 those 喜歡後勤玩法的玩家提供了穩定的任務來源。
*   **聯合開路者俱樂部 (United Wayfarers Club)**：新的聲望體系，專為後勤人員設計。提升聲望可解鎖更高級的燃料合約與特種燃料。

## 3. 行動基地：艦載機庫服務 (Ship Hangar Services T0)

旗艦級船隻（Idris, Polaris, 890 Jump, Carrack）的定位在 4.8 發生了重大轉變，它們現在是真正的「移動行動基地」。

*   **無縫後勤**：只要飛船停靠在這些大型船隻的機庫內，機長即可透過 mobiGlas 直接對停靠的子機進行**加油、修理與補彈**。
*   **戰術價值**：這意味著小隊可以遠離空間站，在敵對區域進行持久作戰。

## 4. 系統改革：保險與物資回收

保險系統在 4.8 進行了拆分，這是一個影響深遠的經濟改動，旨在平衡玩家的風險與收益。

| 保險類型 | 恢復內容 | 成本與等待時間 |
| :--- | :--- | :--- |
| **基礎理賠 (Base Claim)** | 恢復飛船的出廠原始配置 (Stock)。 | **免費**，但申請理賠後的等待時間較長。 |
| **自定義理賠 (Custom Claim)** | 恢復飛船最後一次「印記」(Imprint) 的自定義配置。 | 需支付一定的 **aUEC**，但等待時間大幅縮短。 |

> **專家提示**：每次更換飛船武器或組件後，記得在空間站的終端進行「配置印記」，否則自定義理賠將無法找回你的昂貴裝備。

## 5. 戰鬥平衡：護甲與 G 力

*   **護甲轉向物理化**：大型船隻的厚重護甲現在能更有效地反彈（Deflect）小口徑武器的傷害。這使得「以小博大」變得更具挑戰性，強調了魚雷與重型火炮的重要性。
*   **飛行服與 G 力抗性**：不同的飛行服現在會影響飛行員對 G 力的耐受度。重型抗 G 服能讓你進行更激烈的機動而不會黑視，但會犧牲地面行動的靈活性。

## 6. 新增載具與武器

*   **新船隻**：**Drake Ironclad**（重型運輸與維修）與 **MISC Starlite**（中型專業加油船）正式加入。
*   **新武器**：
    *   **UniFlex 十字弩**：無聲的遠程暗殺武器，適合滲透任務。
    *   **Kastak Arms 等離子手榴彈**：產生持續性的等離子傷害區域，極佳的區域封鎖工具。

---

**結語**：Alpha 4.8 標誌著星際公民向「群體協作」邁出了一大步。無論你是喜歡在前線衝鋒陷陣的戰士，還是在後方默默支持的加油員，這個版本都為你提供了更廣闊的舞台。
