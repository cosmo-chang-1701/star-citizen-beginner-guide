# 0.3：最重要的心理準備：這是一趟充滿驚喜（與 Bug）的 Alpha 旅程

這是我們在出發前，必須反覆強調的最重要的一點：你即將進入的，是一個處於 **Alpha 測試階段**的遊戲。

請把這句話刻在你的腦海裡。這不是一個已經完成、可以順暢遊玩的最終產品。你不是單純的「玩家」，在某種程度上，你也是一名「測試員」。你的每一次登入，都是在幫助開發團隊收集數據，測試這個龐大宇宙的極限。

這意味著什麼？

- **你會遇到 Bug：** 各式各樣、千奇百怪的 Bug。電梯可能會把你吞噬、任務目標可能不會出現、你的飛船可能會在太空中突然爆炸……這些都是 Alpha 旅程的一部分。
- **伺服器與機制的不穩定：** 你可能會遇到物品無法使用、裝備穿不上去，或者 NPC 行為異常。特別是當涉及複雜的專業後勤操作或大規模協作任務時，系統延遲有時會導致預期外的結果。
- **遊戲會崩潰：** 雖然穩定性已大幅提升，但客戶端有時仍會無預警地關閉。
- **進度調整與重置 (Wipe)：** 隨著**持久化技術 (PES)** 的成熟，全面的進度重置已變得越來越少見。現在的調整通常僅影響消耗性物品（如彈藥、醫療針）或進行中的精煉訂單。雖然開發團隊目標在邁向正式版以前盡可能保留玩家資產，但為了平衡經濟或修復重大系統漏洞，數據調整仍是開發旅程的一部分。**（別擔心，你用真實貨幣購買的船艦和裝備是永久的。）**

看到這裡你可能會想：「這聽起來考驗真大！」

但請換個角度思考：正是因為處於 Alpha 階段，這個宇宙才充滿了未知和可能性。你會看到遊戲在你眼前一步步成長、進化。你會因為成功克服挑戰、完成一個困難的戰術任務而獲得無與倫比的成就感。

所以，請務必調整好你的心態：

- **保持耐心：** 遇到問題時，試著重啟遊戲，或者向社群求助。這是一場集體的探索。
- **不要執著於「農」：** 既然經濟體系仍會調整，就不要把賺錢當成唯一目標。去嘗試不同的專業職能，磨練你的戰術技巧。在 Alpha 階段，你的「經驗與技術」才是真正能跨越重置而永久保留的財產。
- **學會變通：** A 計劃行不通？那就試試 B 計畫。這就是擬真宇宙的挑戰與樂趣。

當你接受了這趟旅程的不完美，你才能真正體會到《星際公民》那獨一無二、令人沉醉的魅力。
