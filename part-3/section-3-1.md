在《星際公民》中，跑貨運 (Hauling) 是賺取 aUEC 最穩定也最有利可圖的職業之一。隨著 **貨物電梯 (Freight Elevators)** 與手動裝卸機制的引入，這項職業已從單純的「飛行」演變為一門結合後勤規劃與駕駛技術的專業學問。

## 如何開始貨運生涯？

1.  按下 `F1` 打開你的 mobiGlas。
2.  進入「合約管理器」(Contract Manager) -> 「General」 -> 「Hauling」(貨運)。
3.  **理解聲望體系：** 你的職業生涯由聲望驅動，從「實習生 (Trainee)」起步，目標是成為「大師 (Master)」。
    - **階級順序：** Trainee → Rookie → Junior → Member → Experienced → Senior → Master。
    - **關鍵獎勵：** 提升聲望不僅能解鎖報酬更高的任務，高等級（如 Senior/Master）還能獲得稀有的**藍圖 (Blueprints)**，用於製造軍規級飛船組件。

## 核心機制：貨物電梯與手動裝卸

在 4.8 版本中，貨物不再自動出現在船上。你需要親手完成裝卸：

1.  **呼叫貨物：** 在機庫或哨站的貨物電梯終端機上，選擇「Missions」分頁，將任務貨箱移至電梯平台。
2.  **升起電梯：** 點擊升降按鈕，貨箱會實體化出現在平台上。
3.  **裝載貨物：** 使用 **牽引光束 (Tractor Beam)** 或 **ATLS 動力裝甲** 將貨箱搬移至飛船的貨格 (Cargo Grid) 上。貨物靠近網格時會發出藍光並自動「吸附」。
4.  **卸載與交付：** 抵達目的地後，將貨物搬至電梯平台並降下電梯，最後在終端機點擊「Confirm」完成交付。

## 新手賺錢與晉升建議

### 第一步：基礎積累 (Trainee/Rookie)
- **推薦星系：** Stanton 的 **ArcCorp** 或 **MicroTech**。
- **操作：** 接取太空站往返地表城市（如 Bajini Point ↔ Ryker Memorial）的小型貨運任務（8-12 SCU）。
- **工具：** 基礎牽引光束即可應付，此階段重點在於熟悉電梯操作與飛行流程。

### 第二步：規模擴大 (Junior/Member)
- **推薦路線：** **Hurston 太空站 (Everus Harbor) 循環**。一次接取多個送往地表不同哨站的合約。
- **必備工具：** 強烈建議購買 **ATLS 動力裝甲**，它能極速搬運 32 SCU 的巨型貨箱。
- **推薦船艦：** **Constellation Taurus** 或 **Zeus Mk II CL**，擁有足夠的空間來「堆疊任務」。

### 第三步：星際長途 (Senior/Master)
- **跨星系航線：** 接取從 Stanton (如 Everus Harbor) 前往 Pyro (Ruin Station) 或 Nyx (Levski) 的長途合約。
- **驚人報酬：** 這類任務一趟來回利潤可觀，且能大幅提升聲望，解鎖最頂級的藍圖與合約。
- **推薦船艦：** **C2 Hercules** (目前最穩定的重型運輸船) 或 **MISC Hull-C**。

## 貨運商必學技巧

1.  **任務堆疊 (Stacking)：** 最大化利潤的核心。盡可能在同一個起點接取多個目的地相同的合約。
2.  **分類裝船：** 由於一個任務可能包含多個箱子，裝船時請按目的地分區域堆放，避免在卸貨時搬錯箱子導致任務失敗。
3.  **喚醒終端機：** 在地表哨站操作電梯前，**務必先與終端機互動**。這能確保電梯系統處於活動狀態，避免貨物「消失」的 Bug。
4.  **升級量子引擎：** 對貨運商來說，時間就是金錢。優先更換一個速度最快的量子驅動器，能顯著提升你的單位時間收益。
