# 3.3：採礦：工業玩法的基石

嗨，公民！

歡迎來到採礦的世界。這不僅僅是《星際公民》中賺取 aUEC 的一種方式——它是一門藝術，一條完整的職業道路。無論你是剛拿到 Pyro 多功能工具的新手，還是準備指揮工業巨獸的老手，這裡都有你的一席之地。

坐穩了，接下來我會跟你分享一下這份職業的點點滴滴，從最基礎的到最進階的策略。

### 完整的採礦週期：如何一步步挖到寶

好了，公民，我們來拆解一下完整的流程。

- **階段一：探勘（掃描）**
  按下 `Tab` 鍵啟動掃描模式，長按並釋放滑鼠左鍵發出脈衝。你會在 HUD 上看到藍色方框，那些就是可開採的岩體。

- **階段二：分析與破裂（雷射小遊戲）**
  按下 `M` key 進入採礦模式，對準岩體。你的電腦會告訴你它的**成分 (Composition)**、**質量 (Mass)**、**電阻 (Resistance)** 和**不穩定性 (Instability)**。這一步最關鍵，你要在這裡決定這塊石頭值不值得你花時間。

  決定開採後，開始「破裂」：

  1.  **破裂模式 (Fracture Mode)：** 這是你的主要模式。
  2.  **雷射功率 (Laser Throttle)：** 用滑鼠滾輪控制。
  3.  **目標：** 將「岩體能量水平 (Rock Energy Level)」維持在 HUD 上的**綠色區間**內。
  4.  **平衡：** 你要讓「破裂感應器 (Fracturing Sensor)」在「過載感應器 (Overcharge Sensor)」之前達到 100%。如果過載了，你和你的船就危險了。

- **階段三：萃取**
  岩體成功破裂後，切換到**萃取模式 (Extraction Mode)**（滑鼠右鍵）。

  - **紫色輪廓**的碎片：準備好被吸取。
  - **黃色輪廓**的碎片：太大了，需要再破裂一次。
  - **技巧：** 優先吸取那些礦物含量高、惰性物質少的碎片，最大化你有限貨艙的價值。

- **階段四：精煉（利潤最大化）**
  **永遠不要直接賣原礦！** 把它們帶到太空站的精煉廠。在終端機創建一個**工作訂單 (Work Order)**，然後選擇**精煉方法**。這是一個策略選擇：

| 精煉方法    | 產量 (衰減) | 速度 (倍率) | 適用情境                           |
| :---------- | :---------- | :---------- | :--------------------------------- |
| **Dinyx**   | 0%          | 8 倍時間    | 追求長期收益最大化，不在乎時間。   |
| **Ferron**  | -2%         | 4 倍時間    | 優秀的平衡點，兼顧產量與效率。     |
| **Cormack** | -10%        | 1 倍時間    | 急需用錢，不惜犧牲產量來換取速度。 |

    * **專業化加成：** 有些精煉廠對特定礦物有加成，注意觀察。

- **階段五：後勤與銷售（最後一哩路）**
  精煉訂單完成後，你需要開一艘**貨船**（不是你的採礦船）回到精煉廠，把精煉後的商品轉移到貨船上。
  然後，把這些高價值貨物運到主要城市（如 Lorville 或 Area 18）的貿易與發展部 (TDD) 出售。
  **警告：** 這是風險最高的階段。滿載精煉貨物的你，是海盜的頭號目標。保持警惕！

### 進階策略：老手的心得

公民，掌握了基礎後，這些策略能讓你更上一層樓。

- **Quantanium 的賭局（高風險，高回報）**
  這是 Quantanium 獨有的機制：**它會揮發**。
  一旦你把 Quantanium 萃取進船艙，一個**倒數計時器**就會啟動。你必須在計時器歸零前，將船**安全儲存**在精煉廠。如果失敗，你的船和所有貨物都會炸成一團火球。
  **標準流程：**

  1.  先把整塊 Quantanium 岩體**完全破裂**成可萃取的碎片。
  2.  **然後，再一口氣**把所有碎片吸入船艙（這時計時器才啟動）。
  3.  規劃**最短路線**直飛精煉廠。
  4.  落地後，**第一時間衝向終端儲存船艦**，凍結計時器。
      這是一場與時間的賽跑，也是遊戲中最刺激的體驗之一。

- **最大化你的 aUEC 時薪**
  不要只看單次收入。專業礦工會計算「aUEC 時薪」。這包括了你所有的時間成本：探勘、開採、飛行、精煉等待、運輸和銷售。

  - **善用工具：** 使用社群開發的第三方工具，它們能幫你計算精煉收益、查詢市場價格，找到最佳的銷售地點。

- **MOLE 團隊準則**
  如果你在 MOLE 上，記住：
  1.  **溝通為王：** 持續的語音溝通是成功的基石。建立標準指令（例如，「三號雷射，準備上 Surge」）。
  2.  **角色專精：** 駕駛員的任務是找到最佳停船位置，讓所有雷射砲塔都能開火。操作員則專注於破裂和萃取。

3. **利潤分配：** 在出發前就商量好如何分配利潤。扣除成本後均分，是維持團隊長久的方法。

### 職業的未來

公民，我們現在所做的，不僅是為了積累財富。採礦，是未來**製造 (Crafting)** 和**基地建設 (Base Building)** 系統的經濟基石。

我們開採的礦物，未來將成為製造船艦組件、武器乃至玩家前哨站的關鍵原料。你現在精通的每一項採礦技能，都是在為未來那個玩家驅動的完整工業經濟體系，進行提前投資。

採礦在《星際公民》中不僅僅是一個職業，它是一個複雜、富有挑戰性且對遊戲未來至關重要的核心支柱。

歡迎加入礦工的行列，公民。我們礦區見。
